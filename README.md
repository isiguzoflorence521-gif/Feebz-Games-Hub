# Feebz-Games-Hub
Ultimate VR launcher for Samsung S24 Ultra &amp; Feebz 2.0. Optimizes for QHD+/120Hz to kill the screen door effect. Features S-Pen button mapping, Grayscale AR passthrough (cover-off mode), and "Shared Access" pings for a 25+ game library including Whispering Eons and BattleGun VR. 🚀

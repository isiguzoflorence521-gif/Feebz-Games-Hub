# 🌌 Feebz Games Hub: Ultimate VR

**The definitive VR launcher for Samsung S24 Ultra & Feebz 2.0.**

## 📖 Description
Ultimate VR launcher for **Samsung S24 Ultra** & **Feebz 2.0**. Optimizes for **QHD+/120Hz** to kill the screen door effect. Features **S-Pen button mapping**, **Grayscale AR passthrough**, and "Shared Access" pings for a 25+ game library. 🚀

## 🚀 Key Features
*   **Auto-UHD:** Forces 3120x1440 resolution.
*   **S-Pen Wand:** Maps the S-Pen button as a VR trigger.
*   **Cover-Off Mode:** Optimized for AR recording with the front plate removed.
*   **Shared Access:** Remote pings for new "Full Dive" titles.

## 🎮 Top Games
*   **Space:** *Whispering Eons, VR Space Stalker, Evo VR.*
*   **Action:** *BattleGun VR, The Lost Future, Snow Strike.*
*   **Speed:** *Voxel Fly, VR Xracer, Roller Coaster.*

## 🛠️ Setup
1. Clone this repo.
2. Set S24 Ultra to **120Hz Adaptive**.
3. Remove Feebz front cover for cooling/AR.
4. Launch `FeebzHub.apk`.
